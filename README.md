# Health Insurance Risk Modeling with MEPS

Predicting total annual healthcare expenditures (`TOTEXP`) using MEPS public-use data.

- **Train/validate:** HC-243 (2022)
- **Real-world test:** HC-251 (2023)

## Repo layout
- `notebooks/` — main analysis notebook(s)
- `docs/` — MEPS documentation PDFs
- `data/raw/` — local-only MEPS `.dta` files (not tracked)

## Data note
Raw MEPS data files are not included in this repository.
